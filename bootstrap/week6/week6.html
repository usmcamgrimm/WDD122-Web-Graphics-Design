Bootstrap Week 1

BS 1 BS 2 BS 3

I created the entire page for this course using Bootstrap, as it can be a simple way to create responsive content. However, I feel learning a front end framework in place of the fundamentals is a disservice to students.

PRO:

  • Easy to set up and get started
  • Responsive content by design
  • Consistency through modular components
  • Cross-browser compatibility
  • Open source downloadable components

CON:

  • Bootstrap sites can tend to look similar
  • Necessary code can make websites heavy
  • Can be time-consuming to customize
  • Can cause conflicts with non-bootstrap code
  • Does not always follow "best practices"
