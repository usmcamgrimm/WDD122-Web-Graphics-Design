Bootstrap Week 1

BS 1 BS 2 BS 3

I created the entire page for this course using Bootstrap, as it can be a simple way to create responsive content. However, in this class I'd like more emphasis on design systems, design psychology, creating effecive layouts or color theory. Learning a front end framework in place of the fundamentals is a disservice to students.

PRO:

  • Easy to set up and get started

  • Responsive content by design

  • Consistency through modular components

  • Cross-browser compatibility

  • Open source downloadable components

CON:

  • Bootstrap sites look similar

  • Website can be heavy

  • Can be time consuming to customize

  • Conflicts with non-bootstrap programming

  • Does not follow best practices
